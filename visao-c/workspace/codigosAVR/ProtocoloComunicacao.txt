Inicializador - ST
Tamanho	- SZ - Tamanho do pacote em bytes (ID e DT)
Identificador de comando - ID
Dados - DT
Checksum - CH - Conta tudo menos ST e SZ


ST SZ ID DT CH
7E -- -- -- --


Pacotes de dados
Comando de movimentação de servo
data[0](ID) - Se ID = 01
data[1]     - Adress servo
data[2]     - Porcentagem 

Se ID = 02 (seta limites) - Valores multiplos de 10(?)
D3 D2 D1 D0

Pacote de resposta:

A resposta do pacote é um eco do pacote enviado. A única alteração é o identificador do comando (ID) que tem um acrescimo de 0x7F, e por consequência o checksum.
Exemplo:
Envio:
ST SZ ID DT DT CH	-- Comando de mover o servo 02 para posição 10%
7E 03 01 02 10 EC
Resposta:
ST SZ ID DT DT CH	-- Comando de resposta
7E 03 80 02 10 6D



Se ID = 0x11 Requisita corrente
data[0](ID) - Se ID = 0x11
data[1]     - Adress servo

Resposta:
data[0](ID) - ID = 0x90
data[1]     - Adress servo
// Dado deslocado
data[2]     - 8 bits ADC (2-9)
// Dado inteiro
data[2]     - 8 bits mais significativos do ADC (8-15)
data[3]     - 8 bits menos significativos do ADC (0-7)


Se ID = 0x13 Requisita corrente de todos servos
data[0](ID) - Se ID = 0x13

Respostas:
data[0](ID) - ID = 0x93

data[1]     - Corrente Servo_0
data[2]     - Corrente Servo_1
data[3]     - Corrente Servo_2
data[4]     - Corrente Servo_3
